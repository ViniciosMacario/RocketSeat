# Encapsulamento

Dirigir carro vs conhecer o motor do carro.
  Você não precisar entender toda a estrutura do carro para dirigir.

- Colocar numa cápsula.
- Agrupamento de funções e variáveis.
- Esconder detalhes de implementação.
- Camada de Segurança para os Atributos e Métodos.

Nossos atributos e métodos que estavam espalhados pelo nosso código vão estar dentro de uma classe, ou seja, estamos Encapsulando, escondendo detalhes para o usuário final.

Você pode usar os Objetos no Js sem mesmo conhecer a fundo o funcionamento deles,mas saiba, por de baixo do panos, acontece muito mais coisas.
